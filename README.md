# ZION PROJECTS CONSTRUCTION LTD Website

## Overview

The **ZION PROJECTS CONSTRUCTION LTD Website** is a modern, enterprise-grade Content Management System (CMS) developed to provide a professional online presence for **Zion Projects Construction Ltd**, a Malawian-owned construction and civil engineering company.

The system combines a responsive public-facing website with a secure administration portal, enabling the company to showcase its services, projects, news, and corporate information while allowing authorized staff to manage website content without programming knowledge.

# Installation

1. Clone the repository.

```
git clone <repository-url>
```

2. Import the SQL database.

3. Configure the database credentials in the environment configuration file.

4. Point your web server document root to the project's public directory.

5. Log in to the administration portal.
